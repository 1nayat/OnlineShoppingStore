Online Shopping Store

A web-based e-commerce application built using ASP.NET Core MVC that enables users to browse products, manage a shopping cart, and place orders, with administrative capabilities for managing products and system data.
This project demonstrates core e-commerce workflows using a layered MVC architecture and repository pattern.

Overview
OnlineShoppingStore simulates a complete online retail platform with both user and administrative functionality.
It includes:
Product catalog management
Shopping cart functionality
Order placement workflow
Admin management interface
Repository-based data access

Features
User Features
Browse product catalog
View product details
Add products to cart
Update cart quantit
Remove items from cart
Checkout and place orders

Admin Features
Add new products
Edit existing products
Delete products

Manage product inventory

Styled admin dashboard
